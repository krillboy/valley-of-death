Here's a concise list of basic Markdown syntax elements (tags are actually just plain text formatting):

Headers
# Heading 1
## Heading 2
### Heading 3
Up to
###### Heading 6

Text Formatting
*italic* or _italic_
**bold** or __bold__
~~strikethrough~~
`inline code`
Links and Images
[link text](https://example.com)
![alt text](image-url.jpg)

Lists
Unordered: - item or * item
Ordered: 1. item
Nested: Indent with 2+ spaces or tabs

Code Blocks
```language
code here
```

Blockquotes
> Quote text
Horizontal Rule
--- or ***
Tables
| Header1 | Header2 |
|---------|---------|
| Cell1   | Cell2   |
For more advanced features or rendering, use a Markdown previewer or reference like CommonMark.
